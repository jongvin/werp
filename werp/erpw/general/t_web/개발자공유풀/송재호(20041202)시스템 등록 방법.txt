1. 시스템플랫폼 폴더의 파일들을 COPY해서 해당 시스템 도메인에 PASTE한다.
2. 사용자 관리 시스템에 들어간다.
3. 시스템 메뉴에 들어가서 해당 시스템의 정보를 입력한다.
   1)시스템URL- 시스템 초기 서비스 페이지 URL(반드시 index.jsp로 한다.)
                EX) http://ccms.dihv.co.kr/um/index.jsp
   2)동일 LOGO URL - 상단 동일토건 이미지을 클릭 했을때 이동 할 서비스 페이지명
   3)content url   - 시스템 초기 메인 서비스 페이지명
   4)help url      - 서비스 페이지명
   5)sitemap url   - 서비스 페이지명
