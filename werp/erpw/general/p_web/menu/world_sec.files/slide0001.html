월드진단 2차 평가
개요
  - 평가차수별 월드진단 2차평가를 작성합니다.
① 평가종류와 평가차수를 선택하면 해당 피평가자 리스트가 조회됩니다.
② 피평가자별로 1차평가자 제출여부와 제출일과 제안등급, 2차평가자 제출여부, 평가등급이 조회됩니다.
③ 피평가자 리스트의 공통 체크박스를 클릭하면 피평가자의 월드진단(공통역량) 자기평가, 1차평가 내용이 조회됩니다.
④ 피평가자 리스트의 리더쉽 체크박스를 클릭하면 피평가자의 월드진단(리더쉽역량) 선임부하평가, 1차평가 내용이 조회
    됩니다.
⑤ 평가내용을 확인한 후 닫기버튼을 클릭하여 피평가자 리스트 화면에서 가이드라인 배분율에 맞춰 평가등급을 입력합니
    다. 이 때, 2차 평가자는 공통역량과 리더쉽역량 평가의 결과를 종합하여 평가등급을 입력합니다.
⑥ 평가가 완료된 건은 선택 항목을 체크하고 제출버튼을 클릭하여 평가자료를 제출하면 됩니다.
⑦ 반송이 필요한 건은 선택 항목을 체크하고 반송버튼을 클릭하여 반송할 수 있습니다.
* 월드진단(공통역량)
1차평가자의 제안등급이 있음.
* 월드진단(리더쉽역량)
1차평가자의 제안등급이 없음.
