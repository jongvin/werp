① 평가년도를 입력하면 해당 피평가자 리스트가 조회됩니다.
② 피평가자별로 제출여부와 제출일, 1차평가자 목표합의여부와 목표합의일이 조회됩니다.
③ 피평가자 리스트를 더블클릭하면 피평가자의 MBO 목표수립 내용이 조회됩니다.
④ 피평가자의 MBO 목표수립 내용을 확인하고 반송 또는 목표합의를 합니다.
⑤ 반송을 하면 피평가자는 목표수립을 다시 작성하여 제출하는 방식으로, 피평가자와 1차평가자 사이에 목표수립을
    합니다.
MBO 목표수립 1차
개요
  - 1차평가 피평가자 리스트를 조회하고 각 피평가자별로 목표수립에 대해 합의합니다.
