① 평가차수를 입력합니다.
② 1차평가자와 2차평가자가 자동으로 셋팅됩니다
③ 셋팅이 잘못되었거나 안 되었을 경우에는 찾기 버튼을 클릭하여 사원찾기 팝업 창에서 1차평가자와 2차평가자를 정확
    하게 작성합니다.
④ 평가차수의 평가년도에 수립한 목표가 화면에 조회됩니다.
⑤ 평가항목별로 달성도, 평가의견, 등급을 작성합니다. 점수는 비중에 따라 등급 환산점수로 자동 계산됩니다.
    <점수계산방법> 등급환산점수 * 비중
⑥ 자기평가를 작성한 후에 미달성업무에 대한 조치계획을 작성합니다. 작성하지 않으면 제출할 수 없으므로 미달성업무
    가 없는 경우는 ‘없음’이라고 작성하시면 됩니다.
⑦ 작성이 완료 되었으면 저장버튼을 클릭하여 저장합니다.
⑧ 자기평가가 완료 되었으면 제출 버튼을 클릭하여 1차평가자에게 제출을 합니다.
MBO 자기평가
개요
